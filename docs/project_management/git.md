## Was ist Git

```shell title="Basics"
git init <directory> 

git clone <repository>

git config user.name <name>

git add <directory> 

git commit -m "<message>"

git status

git log 

git diff

```


```shell title="Workflow"

git clone <repository>

git status

git add <directory>

git push

git checkout -b "<branch>"

```
